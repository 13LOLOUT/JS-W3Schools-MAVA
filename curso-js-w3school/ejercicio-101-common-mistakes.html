JavaScript Common Mistakes

let x = 0;
if (x = 10) {  // ❌ Asignación
  console.log("Ejecuta!");
}

Errores más comunes:

  • = en lugar de == en if
  • == loose comparison (usar ===)
  • switch con strings cuando es number
  • + número + string = concatenación
  • Floats: 0.1 + 0.2 != 0.3
  • Semicolon después de if
  • return en línea nueva
