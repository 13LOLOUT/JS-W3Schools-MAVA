JavaScript Breakpoints - F12 Sources

Presiona F12 → Sources → Click números línea → Reload

PASO A PASO Breakpoints:

  1. F12 → Sources (Chrome/Edge)
  2. Encuentra este archivo JS
  3. Click número línea (punto azul)
  4. F5 Reload página
  5. Play línea por línea

Controles Debug:

  • Resume continúa
  • F10 Step Over siguiente línea
  • F11 Step Into entra función
  • Shift+F11 Step Out sale función
  • Scope panel → variables actuales
